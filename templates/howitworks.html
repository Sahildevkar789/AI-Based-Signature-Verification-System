How the System Works

A step-by-step overview of the AI-based signature verification pipeline

Signature Enrollment

Multiple genuine signature samples are collected for each student to learn natural variations in handwriting.

Image Preprocessing

Uploaded signatures are converted to grayscale, binarized, and normalized for consistent feature extraction.

Feature Extraction

Structural and similarity-based features are extracted using keypoint detection and matching techniques.

ML-Based Classification

A trained machine learning model classifies the signature as genuine or forged based on similarity scores.

Final Verification

The system outputs the verification result along with a confidence score for transparency.
