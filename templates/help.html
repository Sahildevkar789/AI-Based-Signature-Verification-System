Common questions about the AI-based Signature Verification System

A genuine signature is written by the actual student and is collected during enrollment. These samples represent authentic handwriting patterns.
A forged signature is written by an unauthorized person attempting to imitate the original signature. The system is trained to detect such fraudulent attempts.
The system uses the CEDAR Handwritten Signature Dataset, a standard benchmark dataset widely used in academic research.
No biometric system can achieve 100% accuracy. This system achieves around 91% accuracy with a low False Acceptance Rate (FAR).
The uploaded signature is preprocessed, compared with enrolled samples using feature matching, and classified by a trained machine learning model.
