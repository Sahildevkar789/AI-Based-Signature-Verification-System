How the System Works

Signature Enrollment

Multiple genuine signature samples are collected for each student during enrollment. This helps the system learn individual writing patterns and natural variations in handwriting.

Feature Extraction

The uploaded signature image is preprocessed using image processing techniques such as binarization and normalization. Key structural and similarity features are then extracted for comparison.

ML-Based Verification

Extracted features are analyzed by a trained machine learning model. Based on similarity scores and security thresholds, the system classifies the signature as genuine or forged with confidence.
